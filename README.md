### Installation

- Clone the repository
  ```
  git clone https://github.com/saurabhnimkande/KlarnaFrontend.git
  ```

### `npm install`

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
